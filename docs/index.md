## Overview

StratoQuant Nexus is an institutional-grade crypto trading engine featuring:

- **Multi-layer Architecture**: L0 (Data) → L1 (Signals) → L2 (Risk) → L3 (Execution)
- **TradingView Integration**: Pine Script webhook execution
- **Risk Management**: Built-in position sizing and portfolio protection
- **Paper Trading**: Safe testing before going live
